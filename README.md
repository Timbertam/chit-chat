# chit-chat
The repository for the project I've built for my amazing brother and my amazing brother only... Chit-Chat!

This is a Sonic Robo Blast 2 mod written in LUA that allows dynamic modification of the level's music, background, light level, weather, among other things like SFX playback, power-up retrieval and more. A very ambitious project for a very beautiful game, but I made it entirely for my brother and nobody else... as he deserves the world and more! Cheers... my amazing brother ;D
